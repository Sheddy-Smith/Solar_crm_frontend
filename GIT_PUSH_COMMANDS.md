# Git Push Commands

Run these commands from:

```powershell
C:\Malwa_Solar_CRM
```

```powershell
git status
git add src/App.jsx src/index.css
git commit -m "Update CRM responsive lead UI and form styling"
git push origin main
```

If push asks to set upstream or branch is not linked, run:

```powershell
git push -u origin main
```
